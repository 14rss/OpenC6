# OpenC6
An Open Source (TheUnlicense) Connect-6 Godot Project
Discord RPC by vaporvee

Connect 6, in this game, is more of a connect 4 type of game but with a 10x10 game board and a winning requirement of 6, rather than 4. The reason I chose 6 is because it makes the games more drawn out and strategic, while still being somewhat managable. I may change the board size as time goes on for balancing.
Connect 6 traditionally is a game similar to Gomoku, played on a (19x19) Go board. This doesn't use the same physics aspect and challenge of my version.
In the future, I plan on making the board size easily scalable to allow anyone who wants to use this project to alter it themselves and maybe even turn it into the classic connect 4 experience. That's all up to you, because this application is fully unlicensed, meaning you are allowed to publish it without changing a single thing!
